# Community-Portal-Website
I created a community portal that could potentially allows for my pupils to stay updated on recent/upcoming events. In this website you get kept up to date about everything, not only that but you can even report/inquiry about anything through a contact form. And I have added a admin portal(login system) that allows admins to manage the inquiries or reports sent through the contact form be website users. To add on to that the super admin can login through the same (login system) to have acces to all informations(customers, admins) with allowance to manipulate data as well. Check out README for more on this.
